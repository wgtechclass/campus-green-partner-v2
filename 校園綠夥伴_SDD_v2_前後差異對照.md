# SDD v2 前後差異對照｜《校園綠夥伴：環境復甦任務》→「淨芽小隊」修正版

> 本文件用於輔助 Claude Code、Codex 或其他 AI Coding 工具理解：本次 SDD 不是重寫專案，而是在現有程式基礎上做增量修改。

---

## 1. 系統定位差異

| 項目 | SDD v1 | SDD v2 |
|---|---|---|
| 主系統名稱 | 校園綠夥伴：環境復甦任務 | 保留 |
| 學生端遊戲名稱 | 無獨立遊戲層 | 新增：淨芽小隊：校園健康冒險 |
| 核心體驗 | 任務回報＋寵物成長 | 校園探索＋步數能量＋垃圾辨識分類＋精靈升級＋地圖解鎖 |
| 設計策略 | MVP 任務系統 | 在 MVP 上升級遊戲感 |
| 開發方式 | 建立基礎架構 | 增量修改，不重建 |

---

## 2. 技術架構差異

| 項目 | SDD v1 | SDD v2 |
|---|---|---|
| 前端 | HTML、CSS、JavaScript | 沿用，新增淨芽小隊 UI 層 |
| 後端 | Google Apps Script | 沿用，擴充 actions 與資料欄位 |
| 資料庫 | Google Sheets | 沿用，新增欄位與 MapNodes 工作表 |
| AI | 教師端手動 prompt | 新增 AI 小隊長提示語與 prompt 模組 |
| 圖像 | 可暫時不用或簡化視覺 | 使用提供角色圖裁切、去背後的正式透明 PNG，放入 `assets/spirits/*.png` |

|---|---|---|

| AI Module | 教師分析 prompt | AI 小隊長：任務提示、分類提醒、成果摘要 prompt |

## 4. 頁面差異

| 原頁面 | SDD v1 | SDD v2 |

## 8. 開發影響程度

| 修改項目 | 影響程度 | 建議處理 |
|---|---|---|
| 名稱與文案 | 低 | 直接改前端常數與畫面文字 |
| 精靈角色資料 | 低～中 | 改 Pets 預設資料與角色卡 UI，`imageStage1`～`imageStage4` 填入正式圖片路徑 |
| 角色圖片裁切去背與容器 | 中 | 必做；輸出 `assets/spirits/*.png`，透明背景、正方形、角色完整置中，前端以 `object-fit: contain` 呈現 |
| 學生端遊戲化視覺 | 中 | 必做；國小學生取向，首頁、角色卡、地圖、小隊基地、守護報告需活潑但可讀，且不得爆版 |
| 能量名稱對應 | 低 | 前端 mapping，不改內部欄位 |
| 任務回報新增欄位 | 中 | 前端表單＋GAS 寫入欄位 |
| MapNodes | 中 | 新增表與前端節點顯示 |
| 小隊基地 | 中 | 強化 growth 頁面 |
| 教師端統計 | 中 | 擴充統計函式與表格 |
| 真實步數 API | 高 | 本階段不做 |
| AI 圖像辨識 | 高 | 本階段不做 |
| 重建資料庫 | 高 | 明確不做 |
